# Product Stock Manager

سیستم مدیریت موجودی ووکامرس جهت اسکن بارکد یا ورود دستی کد محصول برای به‌روزرسانی موجودی.

## ویژگی‌ها

- **اسکن بارکد**: از دوربین گوشی برای اسکن بارکد استفاده کنید
- **ورود دستی**: به‌روزرسانی موجودی با وارد کردن ID یا کد محصول
- **افزایش/کاهش موجودی**: تغییر موجودی به صورت نسبی
- **مدیریت متغیرها**: پشتیبانی کامل برای محصولات متغیر (Variable Products)
- **ویرایش قیمت**: ویرایش قیمت عادی و قیمت حراجی
- **جستجو و فیلتر**: جستجوی سریع محصولات و فیلتر کردن بر اساس وضعیت موجودی
- **رابط کاربری مدرن**: طراحی فعلاً‌روز و قابل استفاده

## نحوه استفاده

### صفحه مدیریت موجودی

1. از منوی ووکامرس، گزینه "مدیریت موجودی" را انتخاب کنید
2. می‌توانید:
   - **اسکن بارکد**: روی دکمه "شروع اسکن" کلیک کنید و بارکد را اسکن کنید
   - **ورود دستی**: ID یا کد محصول و تعداد را وارد کنید و "به‌روزرسانی موجودی" را کلیک کنید
   - **افزایش/کاهش**: تعداد و دکمه افزایش یا کاهش را انتخاب کنید

### صفحه لیست محصولات

1. از منوی ووکامرس، "مدیریت موجودی" > "لیست محصولات" را انتخاب کنید
2. می‌توانید:
   - **جستجو**: از کادر جستجو برای یافتن محصول استفاده کنید
   - **فیلتر**: بر اساس وضعیت موجودی (موجود، ناموجود، موجودی کم) فیلتر کنید
   - **ویرایش**: مستقیماً موجودی و قیمت‌ها را ویرایش کنید
   - **ذخیره**: تمام تغییرات را یکجا ذخیره کنید

## توابع اصلی

### Utility Functions (`includes/functions.php`)

- `psm_convert_persian_to_latin()` - تبدیل اعداد فارسی به لاتین
- `psm_convert_latin_to_persian()` - تبدیل اعداد لاتین به فارسی
- `psm_get_webp_image_url()` - دریافت تصویر WebP با fallback
- `psm_update_variable_product_stock_status()` - به‌روزرسانی وضعیت موجودی محصولات متغیر

### AJAX Handlers (`includes/ajax-handlers.php`)

- `psm_get_all_products()` - دریافت لیست محصولات
- `psm_update_product_details()` - به‌روزرسانی جزئیات محصول
- `psm_search_products()` - جستجوی محصولات
- `psm_get_product_by_id()` - دریافت اطلاعات محصول
- `psm_update_stock()` - به‌روزرسانی موجودی
