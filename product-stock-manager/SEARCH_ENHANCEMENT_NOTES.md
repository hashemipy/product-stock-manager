# بهبود جستجوی محصولات

## خلاصه تغییرات

### مشکل قبلی
- جستجو فقط بر روی 50 محصول اول انجام می‌شد
- اگر محصول در آن 50 تا نباشد، پیدا نمی‌شد
- تمام جستجو از سمت کلاینت (Client-side) انجام می‌شد

### حل پیاده‌سازی شده

#### 1. AJAX Handler جدید: `psm_search_products_global`
- **فایل**: `includes/ajax-handlers.php` (خط 268-370)
- **ویژگی‌ها**:
  - جستجو در **تمام محصولات** در پایگاه داده (بدون محدودیت LIMIT)
  - پشتیبانی از محصولات ساده و متغیر
  - جستجو بر اساس: نام، SKU، ID
  - پشتیبانی از کلمات فارسی و لاتین
  - بازگرداندن اطلاعات کامل محصول شامل موجودی

#### 2. جستجوی سرور‌ساید (Server-side Search)
- **فایل**: `assets/js/product-list-page.js` (خط 357-405)
- **ویژگی‌ها**:
  - استفاده از AJAX برای جستجو روی سرور
  - Debounce 500ms: منتظر تا کاربر تایپ کردن را متوقف کند
  - نمایش تعداد نتایج پیدا شده
  - خطاگیری و پیام‌های مناسب

#### 3. بهبودی‌های UX
- دکمه "بارگذاری 50 محصول بعدی" در حالت جستجو پنهان می‌شود
- وقتی جستجو پاک شود، به حالت عادی بازمی‌گردد
- نمایش تعداد نتایج: "محصولی یافت نشد" یا "X محصول یافت شد"

### تغییرات فایل‌ها

#### 1. `includes/ajax-handlers.php`
```php
function psm_search_products_global() {
    // جستجوی تمام محصولات بدون محدودیت
    // بازگرداندن نتایج کامل
}
add_action('wp_ajax_psm_search_products_global', 'psm_search_products_global');
```

#### 2. `assets/js/product-list-page.js`
- جایگزینی جستجوی Client-side با AJAX
- اضافه شدن debounce برای بهترین کارایی
- اضافه شدن تابع `updateLoadMoreButton()` برای کنترل دکمه

#### 3. `templates/product-list-page.php`
- اضافه شدن `psm_nonce` برای امنیت
- اضافه شدن `psm_ajax_url` برای استفاده در جستجو

### نحوه استفاده

1. کاربر در کادر جستجو می‌نویسد
2. سیستم 500ms منتظر می‌ماند (debounce)
3. AJAX درخواست به سرور ارسال می‌شود
4. سرور **تمام پایگاه داده** را جستجو می‌کند
5. نتایج به صفحه برگردانده می‌شوند
6. دکمه "بارگذاری" خودکار پنهان می‌شود

### مثال‌های جستجو

| جستجو | نتیجه |
|------|------|
| `محصول` | تمام محصولاتی با نام حاوی "محصول" |
| `SKU123` | محصول با SKU = SKU123 |
| `1050` | محصول با ID = 1050 |
| `محص` | تمام محصولاتی شروع شده با "محص" |
